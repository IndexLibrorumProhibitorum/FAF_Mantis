# FAF Cybran Mantis – LEGO/GoBricks Style MOC
A community LEGO-style MOC (My Own Creation) of the Cybran Mantis from Supreme Commander, designed for the Forged Alliance Forever community.

This project recreates the Cybran T1 Assault Bot using standard brick parts (primarily GoBricks / LEGO-compatible). The repository contains everything required to build the model yourself, including design files, instructions, parts lists, and printable packaging.

# Building the Mantis
To build the model yourself:
1. Order the parts  
Use the parts list to order bricks from a supplier such as GoBricks, BrickLink, or Webrick.  
The parts list files can be imported directly into many brick marketplaces.
Some websites might throw an error for part 3680c01 and part 57906. Part 3680c01 can be replaced by any 2x2 'turntable' brick. Part 57906 hasn't been produced by GoBricks in the requried color, but the original LEGO version can be bought seperatly on bricklink.

2. Download the instructions  
Use the PDF instruction manual in: `/instructions/mantis_build_manual.pdf`

3. Assemble the model  
Follow the step-by-step instructions to complete the build.

4. Optional: print the box  
The repository includes a printable box design. To reproduce the box:  
Print mantis_box_print_A3.pdf on A3 cardstock  
Cut along the outer lines  
Fold along the marked folds. Folding of cardstock is easier and cleaner if the marked foldlines are lightly scored with a knife before folding.  
Do a dry-fit, then glue the tabs  

5. Optional: print the booklet  
You can also print the instruction manual as a small booklet if desired. The booklet was designed for each page to be A6, though it can naturally be printed larger or smaller. The files include a simple PDF with one booklet page per document page.  
Additionally, I've included a version where the pages are imposed (placed) in such a way that they can be printed on 3 double-sided A4 sheets (mirrored over the long/vertical edge for double sided printing). After printing, the pages should then be cut in half width-wise, and can then be folded in half, sorted, and stapled.
